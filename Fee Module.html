Fee Module

This document covers the complete workflow for managing school fees in the MS Hifz Academy ERP — from setting up fee types and class structures to collecting payments, applying concessions, and generating fee reports.

Navigate to Financial Module

  1. Log in to the ERP portal and select the required Branch from the header.
  2. In the left sidebar, click on Financial.
  3. The Financial module will open displaying the fee management menu.
ⓘ Full access to the Fee Module requires the Admin or Finance Manager role. Class Teachers have read-only access to fee information.
Fee Module — Main interface for managing school fees.
Step 1 — Create Fee Types

Fee Types are the categories of fees your institution collects (e.g., Tuition Fee, Transport Fee, Exam Fee).

  1. In the Financial module, click on Fee Types.
  2. The Fee Type Master screen will open, showing existing fee types.
  3. Click Add Fee Type.
  4. Enter the Fee Type Name (e.g., "Tuition Fee", "Transport Fee", "Library Fee").
  5. Enter a short Description (optional).
  6. Set the Installment or Standard type to Active.
  7. Click Save. The fee type is now available for class fee structures.
ⓘ Fee Types are created at the branch level. We can copy fee types from one branch to another. I all branches share the same fee type catalog.
ⓘ Do not delete a fee type once it has been used in a class fee structure or student assignment. Deactivate it instead by toggling its Status to Inactive.
Create Fee Type — Interface for adding new fee types.
Step 2 — Configure Fee Installments

Installments define the payment schedule — which months fees are due and due dates.

  1. In the Financial module, click on Fee Installments.
  2. Select the Academic Year and Branch.
  3. The installment setup shows the months from May to April (standard academic year cycle).
  4. For each month, set:
    • Installment Name — e.g., "May Installment", "June Installment".
    • Start Date — The date when the installment becomes active.
    • Last Date — The date by which payment is expected..
    • Cut-off Date — The date before which the installment can get rebeate .
    • Include in Fee — Toggle whether this month is billed.
  5. Click Generate Bulk Installments to create mulitple fee installments.
  6. Click Save Installments.
ⓘ The installment schedule is created for all classes in a branch for a given academic year. Students get their dues month-by-month based on this schedule.
Fee Installments — Interface for configuring payment schedules.
Step 3 — Set Up Class Fee Structure

Define the base fee amounts for each class and fee type.

  1. In the Financial module, click on Class Fee Structure.
  2. Select the Academic Fee or Existing Student Fee.
  3. Click on Add Fee
  4. The screen will show drop down of Fee Types.
  5. For each fee type combination, enter the Total Fee Amount (in Rupees).
    • Example: Class 6 Tuition (₹15,000) ÷ 6 Installments = ₹2,500/month.
  6. After entering all amounts, click Save Fee Structure.
ⓘ The class fee structure serves as the default template. Individual student fees can be customized after the structure is set up.
ⓘ You can copy the fee structure to another branch by clicking Copy to Branch, selecting the target branch, and confirming.
Create Class Fee — Interface for setting up class-specific fee structures.
Step 4 — Assign Fees to Students

After setting the class fee structure, assign fees to individual students.

Auto-Assignment (Recommended)

  1. When a new student is enrolled (via Student Administration → Create Student), the system automatically assigns the class fee structure to the student.
  2. No manual action is required for standard students.

Manual Assignment

  1. In the Financial module, go to Take Fee and search for the student.
  2. If the student does not have fees assigned, click Assign Standard Fee.
  3. The system will apply the class fee structure for the student's class.
  4. Click Save.
ⓘ If a student is admitted mid-year, only the months from the admission month onward are billed. Previous months are automatically excluded.
Step 5 — Apply Concessions (Discounts / Waivers)

5A — Create a Concession Master

  1. In the Financial module, click on Concession Master.
  2. Click Add Concession.
  3. Enter:
    • Concession Name — e.g., "Sibling Discount", "Merit Scholarship", "Staff Ward Waiver".
    • Concession Amount — Enter the flat amount (₹).
    • Concession Dropdown — If it is Rebate concession check on fee payment dropdown.
    • Applicable Fee Type — Select which fee type this applies to (Tuition, Transport, All).
  4. Click Save. The concession is now available to apply to students.
Create Concession — Enter details for the new concession.

5B — Apply Concession to a Student

  1. In the Financial module, click on Student Concession.
  2. Search for the student by name or admission number.
  3. Select the student from the results.
  4. Click Apply Concession.
  5. Select the Concession from the dropdown (created in 5A).
  6. Select the Fee Type to which the concession applies.
  7. Enter the Effective From Month (e.g., June 2025).
  8. Click Save. The concession will be reflected in the student's fee dues from the selected month.
ⓘ Multiple concessions cannot beapplied to a single student (e.g., sibling discount + merit scholarship). The system calculates the combined discounted fee automatically.
Apply Concession — Select the concession and fee type for the student.
Step 6 — Assign Special Fees

Special fees are one-time or non-recurring charges (e.g., Admission Fee, Uniform Fee, Exam Registration).

  1. In the Financial module, click on Assign Special Fee.
  2. Search for the student.
  3. Click Add Special Fee.
  4. Select the Fee Type (create a special fee type in Fee Types if not already available).
  5. Enter the Amount.
  6. Enter a Description / Remark (e.g., "Annual Day Contribution").
  7. Set the Due Date.
  8. Click Save. The special fee will appear in the student's outstanding dues.
Assign Special Fee — Enter details for the new special fee.
Step 7 — Collect Fee Payment (Take Fee)

The Take Fee screen is used daily by the accounts staff to record student fee payments.

  1. In the Financial module, click on Take Fee.
  2. Search for the student using their Name or Admission Number.
  3. Click on the student's name from the search results.
  4. The Student Fee Details screen will open showing:
    • Fee Dues Month-wise (Pending installments)
    • Special Fees pending
    • Previous payment history
    • Total Outstanding Amount
  5. Select the installment(s) or fees the student is paying.
  6. Enter the Amount Paid.
  7. Select the Payment Mode:
    • Cash
    • Online / UPI
    • Bank Transfer
    • Cheque
  8. Enter the Payment Date (defaults to today).
  9. Enter any Remarks or reference number (optional).
  10. Click Collect Payment.
  11. A Fee Receipt is automatically generated with a unique receipt number.
ⓘ A receipt is generated immediately after clicking Collect Payment. Use the Print Receipt button to print it for the student/parent.
Collect Fee — Enter payment details for the student.
Step 8 — View Fee Receipt
  1. After collecting payment, click Print Receipt on the confirmation screen.
  2. To view a past receipt:
    • Go to Take Fee → search the student → open their profile.
    • In the payment history section, find the Print/Cancel and click View Receipt.
  3. The receipt includes: Receipt Number, Student Name, Class, Payment Date, Amount Paid, Mode of Payment, Fee Type breakdown, and School Seal / Signature area.
  4. Click Print to print the receipt. Set Scale to 100% and Paper Size to A4.
View Fee Receipt — Details of the collected fee payment.
Step 9 — Cancel / Delete a Payment
  1. Open the student's fee profile via Take Fee → Search Student.
  2. In the Payment History section, locate the Print/Cancel option.
  3. Click the Delete / Cancel icon next to the payment.
  4. Enter a Reason for Cancellation (mandatory).
  5. Click Confirm.
  6. The payment is reversed and the student's dues are updated accordingly. A cancellation audit entry is created.
⚠ Warning: Payment cancellation is permanent and cannot be undone. Ensure the correct payment is selected before confirming cancellation. Only Admin and Finance Manager roles can cancel payments.
Cancel Fee — Select a payment to cancel and enter a reason.
Update Student Fee — Modify the fee structure for a specific student.
Step 10 — Update Student Fee Structure

If a student's class changes or fee revision is needed mid-year:

  1. In the Financial module, click on Update Student Fee Structure.
  2. Search for the student.
  3. The current fee structure assigned to the student will display.
  4. Modify the Amount for any fee type as required.
  5. Set the Effective From Month for the change.
  6. Click Save. Future invoices will reflect the updated amount from the selected month.
ⓘ Past payments are NOT affected by fee structure updates. Only future unpaid installments are adjusted.
Step 11 — Fee Reports

The Fee Reports section provides analytics on fee collection and dues.

  1. In the Financial module, click on Fee Reports.
  2. Select the type of report:
    • Today's Collection — All payments received today across the selected branch.
    • Daily Fee Report — Collections for a specific date range.
    • Monthly Fee Report — Month-wise collection summary.
    • Fee Due Report — Students with outstanding dues (can filter by class, month).
    • Concession Report — Summary of all applied concessions.
  3. Set the required filters (Branch, Academic Year, Class, Date Range).
  4. Click Get Report.
  5. Results are displayed in a table. Click Download Excel or Print to export.
Fee Reports — Summary of fee collections and dues across selected branches.
Late Fee Configuration

The system supports automatic late fee charges for overdue installments.

  1. Go to Setup Your School → Fee Configuration (or Financial module configuration).
  2. Enable Late Fee.
  3. Set the Grace Period (number of days after due date before late fee kicks in).
  4. Set the Late Fee Amount — either a flat amount (₹) or a percentage of the due amount per month.
  5. Click Save.
  6. When a student pays after the due date + grace period, the late fee is automatically added to the total amount due on the Take Fee screen.
ⓘ Late fees can be waived for a specific student at the time of payment by clicking Waive Late Fee on the Take Fee screen (Admin / Finance Manager only).
